# 如何获取注册验证码

注册 ShareSDU 时，验证码会发送至你的**山大云邮**。若尚未开通或不知如何查看，可按下列步骤操作。

## 1. 登录信息服务平台

在浏览器中打开 [山东大学信息服务平台（点击访问）](https://service.sdu.edu.cn)，使用学校统一身份认证账号登录。

![](https://sharesdu.com/doc/imgs/sduservice_login.png)

## 2. 进入山大云邮

登录成功后，在首页找到并进入 **山大云邮**（校园邮箱）入口。

![](https://sharesdu.com/doc/imgs/service_page_email.png)

## 3. 查看 ShareSDU 发来的邮件

进入邮箱后，在收件箱中查找发件人或标题中包含 **SHARESDU** 的邮件，打开即可查看验证码。

![](https://sharesdu.com/doc/imgs/how_to_get_code.png)


> 如果因其他原因无法使用校园邮箱注册，可发送邮件至admin@sharesdu.com、加入QQ群或者私信公众号获取邀请码
