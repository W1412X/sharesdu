# 参与维护

项目欢迎新的维护者参与开发和维护工作！

## 如何参与

### 前端开发

**GitHub 仓库**：[https://github.com/W1412X/sharesdu](https://github.com/W1412X/sharesdu)

参与前端开发的方式：
- Fork 项目仓库
- 创建功能分支进行开发
- 提交 Pull Request
- 参与 Issue 讨论和 Bug 修复

### 后端开发

**GitHub 仓库**：[https://github.com/zzysssigm/SHARE_SDU](https://github.com/zzysssigm/SHARE_SDU)

参与后端开发的方式：
- Fork 项目仓库
- 创建功能分支进行开发
- 提交 Pull Request
- 参与 API 设计和优化

## 联系方式

## 参与方式

1. **通过邮件联系**：发送邮件到 admin@sharesdu.com，说明您想要参与的方向（前端/后端）和具体贡献计划

2. **加入QQ群**：加入 QQ 群 246680702，与其他开发者和维护者交流讨论

3. **GitHub 参与**：
   - 前端：在 [前端仓库](https://github.com/W1412X/sharesdu) 提交 Issue 或 Pull Request
   - 后端：在 [后端仓库](https://github.com/zzysssigm/SHARE_SDU) 提交 Issue 或 Pull Request

## 贡献方向

我们欢迎在以下方面进行贡献：

- **功能开发**：新功能的设计和实现
- **Bug 修复**：发现并修复项目中的问题
- **代码优化**：性能优化、代码重构
- **文档完善**：补充和更新项目文档
- **测试**：编写和优化测试用例
- **多平台适配**：原生应用、小程序等平台开发

期待您的参与！
