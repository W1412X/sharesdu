# 平台开发 - 介绍

## 项目概述

ShareSDU 是一个面向大学生的分享、交流与学习平台，支持多平台访问：

- **Web 端**：响应式设计，支持桌面端和移动端
- **Android 应用**：基于 WebView 的原生应用
- **HarmonyOS 应用**：基于 Web 组件的原生应用

## 技术栈

- **前端**：Vue 3 + Vuetify 3
- **后端**：Django REST Framework
- **跨平台方案**：响应式 Web 设计 + WebView 封装
