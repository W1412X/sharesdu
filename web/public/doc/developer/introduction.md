# 开发者文档 - 介绍

欢迎来到 ShareSDU 开发者文档！本文档旨在帮助开发者了解项目、参与开发、维护和贡献。

## 文档概述

开发者文档分为以下几个主要部分：

### 1. 平台开发

**平台开发**部分包含平台相关的开发文档：

- **介绍** - 平台开发概述、技术栈、参与方式
- **参与维护** - 如何参与项目开发和维护工作
- **平台适配** - 多平台应用开发指南（原生应用、小程序等），包含帖子、文章等内容的解析和构建逻辑
- **数据安全** - 安全漏洞报告和参与安全开发
- **其他工作** - 网站管理、运营、宣传等工作

**适用人群**：前端/后端开发者、移动应用开发者、小程序开发者、安全研究人员、项目维护者

### 2. 微服务开发

**微服务开发**部分包含微服务相关的开发文档：

- **介绍** - 微服务开发概述和支持的类型
- **微服务开发形式** - 技术栈、开发要求和流程
- **身份认证** - 身份认证接口说明
- **数据共享** - 数据共享和帖子发布说明

**适用人群**：学生开发者、微服务开发者、Web 应用开发者

### 3. 其他

**其他**部分包含其他类型的参与方式：

- **其他** - 网站管理、运营、宣传等工作

**适用人群**：网站管理员、运营人员、宣传人员

## 快速开始

### 想要参与平台开发？

查看平台开发相关文档，了解项目概况、参与方式、多平台开发和安全相关参与方式。

### 想要开发微服务？

查看微服务开发相关文档，了解微服务开发、技术要求、认证接口和数据共享方式。

### 想要参与其他工作？

查看其他文档，了解网站管理、运营、宣传等工作。

## 贡献指南

无论您是开发者、安全研究人员，还是运营人员，我们都欢迎您的参与！

- **代码贡献**：提交 Pull Request，参与功能开发和 Bug 修复
- **安全贡献**：报告安全漏洞，参与安全开发
- **文档贡献**：完善文档，帮助其他开发者
- **运营贡献**：参与网站管理、运营和宣传

期待您的参与，共同为大学生提供更好的分享、交流与学习平台！
