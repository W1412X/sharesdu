# 微服务开发 - 介绍

## 概述

ShareSDU 支持学生开发者通过 Web 开发便于学生生活的微服务，这些微服务可以基于学校的学习、生活等场景，为学生提供便利的服务。

## 支持的微服务类型

我们欢迎开发各种类型的微服务，例如：

- **二手市场**：学生物品交易平台
- **天气服务**：校园天气查询和提醒
- **搭子寻找**：学习、运动、娱乐等搭子匹配
- **课程表查询**：课程信息查询和管理
- **图书馆座位**：图书馆座位查询和预约
- **食堂菜单**：食堂菜单查询和推荐
- **其他**：任何有助于学生学习和生活的服务

## 核心功能

### 1. 身份认证

我们提供身份认证接口，返回用户的唯一标识，方便开发者进行用户管理。详情请查看 [身份认证](authentication.md) 文档。

### 2. 数据共享

微服务平台的内容可以选择在本站进行数据共享：

- **新建板块**：为您的微服务创建专门的板块
- **发布帖子**：在已有板块下发布帖子展示微服务内容

详情请查看 [数据共享](data_sharing.md) 文档。

### 3. 帖子格式

支持两种帖子格式：

- **普通文本帖子**：标准的文本格式帖子
- **HTML 帖子**：可以自定义 HTML 样式，优化展示效果

详情请查看 [数据共享](data_sharing.md) 文档中的帖子格式说明。

## 开发流程

1. **联系管理员**：通过邮件或 QQ 群联系，说明您的微服务想法
2. **获取 API 文档**：获取身份认证和数据共享相关的 API 文档
3. **开发微服务**：使用 Web 技术开发您的微服务
4. **集成认证**：集成身份认证接口，获取用户唯一标识
5. **数据共享**：选择合适的方式在本站共享微服务内容
6. **账号配置**：配置用于发布帖子的账号（可新建或使用已有账号）
